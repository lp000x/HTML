Hierarquia de Tabelas

Todas as tabelas em HTML seguem uma hierarquia:

  • TABELA (table)
  • TABELA > Linha de Tabela (table row)
  • Tabela > linha de tabela > cabeçalho (table header)
  • Tabela > linha de tabela > cabeçalho (table header) > dados da tabela (table data)

Ou seja:

  • Tabela é o principal e geral
  • Linhas de tabela delimitam as linhas horizontas da table (o segundo elemento)
  • Cabeçalho da tabela

Tags da tabela:

  • table: adiciona a própria tabela
  • tr: adiciona as linhas horizontais da tabela (tr = table row)
  • td: adiciona os dados desejados (td = table data)

Teste de Tabelas

A1 A2 A3
B1 B2 B3
C1 C2 C3
D1 D2 D3

Semanticamente, em HTML, essa é a tabela. Não está com "cara de tabela", pois isso se faz em CSS.

Estilização geral

Uma vez entendendo a hieraquia e qual tag selecionada no CSS altera qual coisa na tabela, eu posso fazer o que quiser:

  • tr selecionado no css vai alterar toda a linha horizontal;
  • td vai alterar todos os dados (ou seja, todos os quadrados) ou apenas aquele que eu selecionar com class ou id;

Adicionais

  • Se a tabela não for construida seguindo a hierarquia, qualquer dado vai aparecer antes ou depois da tabela
  • td e tr não precisam da tag de fechamento (assim como li)
