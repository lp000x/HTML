Scope: groups

Para definir um scope que está mesclado, é preciso utilizar o colgroup (se for coluna) ou rowgroup (se for linhas). Por exemplo, na tabela abaixo "Nome" é o titulo de apenas 1 coluna, mas "Filme" é um titulo para 3 colunas. No caso do "Filme", é preciso usar o colgroup, pois se eu usar apenas "col" vai contabilizar só a primeira das três colunas. A mesma coisa para "Mulheres" e "Homens", é preciso usar rowgroup.

  • Se a mesclagem de um th estiver abrangendo mais de 1 coluna ou mais de 1 linha, se usa scope="...group", pois é um grupo de colunas ou linhas que esse th está abrangendo
  • Porém se tiver mesclagem mas ela continuar apontando só para 1 linha ou coluna, se usa o scope normal
