Como Declarar Um Documento HTML 5

Se for um editor de código, provavelmente ele cria sozinho apenas com 1 comando. No caso do VS CODE, basta digitar ! que ele cria toda a estrutura do documento. Mas, ainda que exista uma forma automática, é importante entender o que significa cada parte.

Significado DOC

Um documento em HTML é composto por 7 tags únicas e, contando com as de fechamento do head e body, são 9:

  • DOCTYPE
  • HTML
  • Head
  • Meta
  • Meta
  • Title
  • Head de novo, para fechar os metadados
  • body
  • body de novo para fechar o conteúdo

Além disso, algumas dessas tags tem atributos:

  • Html tem lang=""
  • Meta tem charset=""
  • O segundo meta tem name="" e viewport=""

DOCTYPE

>!DOCTYPE<

Doctype é uma declaração (não uma tag) que declara o tipo e versão do documento HTML. Ela se comunica diretamente com o navegador, "dizendo" que todo o documento é um HTML.

Para declarar o HTML, após o >!DOCTYPE< é colocado html.

HTML= lang

Essa tag serve para definir e comunicar ao navegador o idioma padrão do documento. Através do atrbuto "lang" se define o valor, ou seja, a sigla do idioma.

Head

O head é a parte onde são inseridos metadados sobre o site.

Meta charset="UTF-8"

A tag meta é utilizada para inserir metadados no documento. Charset é o atributo que define e informa ao navegador como ler os caracteres devem ser interpretados. UTF-8 é o padrão mais recente que engloba vários caracteres especiais e garatam maior compatibilidade. Sem essa definição, alguns caracteres não são exibidos ou são exibidos de forma quebrada.

meta name e content

Esses 2 atributos são utilizados para manter a responsividade do site.

Viewport é toda a área visivel do dispositivo utilizado para ver o site. Ele é relativo, então se adapta aos diferentes tamanhos de área visivel (celulares, tablets, computadores...).

Content=width=device-width diz que o site ocupará toda a largura do dispositivo e initial-scale=1.0 que site será carregado em zoom padrão (100%).

Title

É onde se coloca o nome do site, aquele que aparece na aba.

Body

Em body é onde o site é construído em si. Todo contéudo é criado entre a tag de abertura e fechamento do body
