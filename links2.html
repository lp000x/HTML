target:_blank

Esse atributo faz com que, ao clicar em um link, a página de destino seja aberta em uma segunda aba.

Atributo Rel

Este atributo indica a relação da página atual com a página de destino. Não é utilizado mais pelo Google para indexação, mas ainda é útil para algoritmos entenderem a relação entre as páginas. Rel, como dito, é o atributo, e o que vier especificando sua relação é um valor. Dessa forma, não é algo para usuários, afinal o rel e seu valor nem sequer são exibidos na rela, mas sim para algoritmos.


Rel: Prev

Prev indica que a página para onde o link está apontando é uma anterior a atual.

Rel: Next

Next indica o contrário de Prev: a página para onde o link está apontando é a próxima em relação a atual.

Rel: External

External indica que a página para onde o link direciona é um destino externo.
