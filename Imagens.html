Como inserir Imagens

Para inserir imagens, é utilizada a tag img. Mais explicações sobre a tag vem na ultima parte (Tags para imagens).

É possível usar diferentes URL e caminhos como src:

  • Arquivos locais (C:/)
  • URL locais (file://)
  • URL remotas (https)
