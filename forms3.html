Requied: min-lenght e max-lenght

Muitos sites definem, no campo de usuário e senha, que esses dados precisam ter um determinado número de caracteres, e enquanto eu não atinjo esse número de caracteres, ele não permite avançar. Como fazer isso?

Simples: basta usar o required e min-lenght, para definir um número minimo de caracteres, e max-lenght para o número máximo.

Se os dados estão certos ou não é a linguagem escolhida que vai definir, o HTML apenas vai colocar uma quantidade minima e/ou um limite máximo e barrar caso esses critérios não sejam atendidos.

Size

Caso eu queira alterar o tamanho da caixa, eu posso usar o atributo size. Na verdade, o size não define o tamanho da caixa, ele define quantas letras serão exibidas na caixa e qual o tamanho da caixa é necessário para exibir a quantidade definida.

Autocomplete

Quando eu vou digital algo em um formulário na internet, geralmente o navegador me dá a opção de autocompletar com meus dados. Isso acontece porque o navegador salva as minhas informações. Mas como fazer isso no meu formulário?

  • Primeiro, é preciso deixar, na tag form, o autocomplete ligado (on).
  • Depois, agora nas tags de input, é preciso usar o autocomplete também, mas especifico para o campo em questão.

Para o navegador sugerir o autocomplete, ele precisa saber o que aquele campo está pedindo para que ele possa usar suas informações salvas. É o autocomplete com seu valor aqui no HTML que eu digo para o navegador qual informação o input está pedindo. Existem várias e várias opções de dados.
