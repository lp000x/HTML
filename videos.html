Vídeos Locais

Para inserir algum vídeo de origem local, basta usar a tag video. Porém, para que o vídeo funcione é preciso adicionar o atributo controls dentro da tag, além de algumas configurações de tamanho de acordo com o layout definido.


Vídeos Remotos

Para colocar vídeos que nãoe estejam na minha maquina, é necessário usar um iframe. A tag video não aceita URLs, apenas arquivos. Por isso não é possível adicionar uma URL de algum vídeo do youtube (ou qualquer outro lugar) no src

O iframe se encontra na parte de compartilhamento do vídeo das plataformas que oferecem essa opção, na parte chamada "Incorporação"

Porém, por algum motivo, sempre dá esse erro com vídeos incorporados pelo youtube.

Facebook também não. Enfim, é um erro a ser consertado.
