Estilização de Colunas

Caso eu queira estilizar uma coluna em especifico, ou colocar um estilo diferente para cada uma, eu tenho que criar uma class e aplicá-la sobre cada item da coluna. Em tabelas pequenas isso até é possível, mas e em tabelas com dezenas ou centenas (ou mesmo milhares) de linhas?

A1 A2 A3 A4
B1 B2 B3 A4
C1 C2 C3 A4
D1 D2 D3 A4

Para fazer isso, o HTML trouxe uma tag nova: colgroup. Essa tag seleciona todas as colunas de uma tabela automaticamente.

Existe ainda um outro recurso: span
