Elementos Sem Input

Até agora foram apresentados vários e vários elementos, todos partindo do input. Mas existem alguns que funcionam sem ele.

Select

O select é aquele recurso que mostra uma série de opções ao clicar nele.

É possível fazer da forma acima, apenas colocando as opções, ou eu posso fazer agrupamentos. Para fazer agrupamentos, eu envelopo todas as opções do mesmo grupo com a tag optgroup, usando também o atributo label

Data List

Caso eu queira fornecer uma lista com dados já definidos para o usuário escolher, mas também oferecer a opção para que ele digite se o que não corresponde existir, eu posso usar input + a tag data list

Para que funcione, eu preciso adicionar, no input associado a datalist, o atributo list="", e dentro de list eu colodo o id definido no datalist

Textarea

Para criar um campo onde o usuário pode escrever algo e enviar, eu posso usar a tag textarea

Para alterar o tamho da caixa de mensagem, eu uso os atributos rows="" (quantas linhas vão ter) e cols="" (quantas colunas vão ter).


Vale dizer: rows e cols não definem um limite de caractere, apenas o tamanho do quadro que será exibido.

Outra coisa: ao usar esse recurso de textarea, o ideal é usar o método post, pois ele não tem limite de caracteres, enquanto que o get tem limite de 3000 caracteres
