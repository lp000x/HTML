Em HTML existem as listas ordenadas e desordenadas.

  • Listas desordenadas não possuem numeração e nem letras, de forma que não existe sequência nos itens listados
  • Sua tag é ul
  1. Já as listas ordenadas possuem uma ordem númerica ou alfabética, de forma que os itens listados seguem uma sequência.
  2. Sua tag é ol

Para ambas as listas, é preciso usar a tag li, de list, para adicionar itens à lista.

Personalização

Caso eu queira alterar a marcação da lista, eu uso o atributo type="" dentro da tag de abertura. Porque não style, já que é uma estilização? Pois é não é uma estilização, é uma alteração na exibição da lista enquanto tag.

Para as OL, eu tenho as opções: Númerica, alfabética caixa alta, alfabética caixa baixa, romana caixa baixa e romana caixa alta.

  1. Númerica
  1. Alfabética caixa alta
  2. Caixa
  3. Alta
  1. Afabética caixa baixa
  2. Alfa
  3. Bética
  1. Romana Caixa Alta
  2. Roman
  3. Roma
  1. Romana Caixa baixa
  2. Roman
  3. Roma
  4. ROM

Caso eu queria alterar onde a sequência da lista começa, é preciso adicionar o atributo: start="" e em start eu coloco a partir de onde quero que comece.

  1. 1
  2. 2
  3. 3
  4. 4

A mesma lógica vale para algarismos romanos e letras. Se eu colocar em start="5", ele vai começar a partir da quinta letra do alfabeto ou do quinto algarismo romano.

  1. a
  2. b
  3. c
  4. d

Para as UL, eu tenho as opções: disc (o circulo preto), circle (circulo branco) e square (quadrado preto)

Listas Mistas

Lista mistas são listas dentro de listas:

  1. Listas
    • Dentro
      1. de
        • listas

Um detalhe que esqueci de mencionar: a tag li é autofechável, então não precisa colocar 2 li, basta um só (por mais que o VS CODE automaticamente coloque 2, funciona do mesmo jeito).
