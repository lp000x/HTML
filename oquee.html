Significado Das Siglas

HTML significa Hypertext Markup Language, ou Linguagem de Marcação de Textos. Marcação de texto é o uso das tags para posicionar e indicar um texto (ou outro elemento) em uma página web. HTML é a linguagem que permite isso.

Não É Linguagem de Programação

Apesar de ser chamado de "Linguagem de Marcação", HTML não é linguagem de programação. Em HTML não há lógica nem funções, apenas tags e atributos que dizem o que é cada elemento da tela, onde deve estar e como se mostrará para o usuário (ou para os algoritmos dependendo de qual tag e onde é colocada).

Seu Propósito

O HTML serve para a criação de páginas na web, fundamentalmente no client-side. Apesar de não fazer tudo sozinho, pois a criação de páginas na web também envolve estilização (e essa parte vem do CSS), o HTML cria toda a estrutura e funcionalidades básicas de uma página. Tag por tag, texto por texto, elemento por elemento, HTML define onde cada elemento vai estar e como será mostrado ao usuário. Não só isso, como também diz, na parte de head, como os algoritmos devem se comportar com a página.
