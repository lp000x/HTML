O Que São Links Internos

Links internos criam comunicação entre páginas dentro de um mesmo site através da ancoragem. Por exemplo:

  • Um artigo de pode redirecionar para outro artigo que apresente um conteúdo relacionado
  • Nos menus de navegação são utilizados links internos para mandar o usúario para os locais selecionados
  • Páginas de contato, sobre nós, politica e outros usam links internos

Como Adicionar Links Internos

Para adicionar links, sejam internos ou externos, é utilizada a tag a com o atributo href (hypertext reference): <a href:>TEXTO ANCORA<a>. Uma vez que o uso é voltado para linkagem interna, o link utilizado em href será de uma página dentro do site.

Ancoragem

Ancoragem é a palavra ou frase (ou outro elemento) sobre a qual o link será aplicado.

Neste exemplo, o link é a tag <a href:>, categorizando um link interno porque dentro de href, obviamente, eu coloquei um link que leva para dentro do meu site.

Os textos "Ver Atributos Padrão" e "Ver Links Externos" são a ancoragem, pois foram aplicados sobre eles a tag <a href:>.

  • Link = tag a href
  • Ancoragem = texto ou outro elemento que recebe o link
