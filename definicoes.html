O Que É HTML

HTML, como já dito, signfica HyperText Markup Language. É utilizado para definir o que é cada elemento na página (seja para usuários ou algoritmos) e onde estará posicionado.


Diferente do CSS, o HTML não faz a estilização do conteúdo, apenas o insere, criando a estrutura. Caso eu desafaça o link com o estilo padrão em CSS nessa página, cor de fundo, centralização, bordas, sombras das caixas... tudo isso vai desaparecer, restando apenas os textos.E diferente de Java, HTML por si só não se comunica com banco de dados e nem realiza animações mais complexas.

O Que É Java

Java, por sua vez, é uma linguagem de programação. Na parte de desenvolvimento frontend é com ele que se faz animações e adiciona outros tipos de recursos na página, como noticias que se alternam na página inicial ou uma série de produtos que são exibidos em ecommerces, até mesmo a contagem de visualizações e likes. O Java não cria nem o estilo nem a estrutura, ele cria a lógica que faz a página deixar de ser estática e responder a interações e inputs.

O Que É CSS

O CSS cria toda o estilo do site. Posicionamento, tamanhos, larguras, cores, até mesmo a estrutura por dentro de cada box (margin, padding, outline...), tudo isso é feito através de CSS. Porém, diferente do HTML que até exibe alguma coisa sem CSS, ele próprio é muito mais limitado sem HTML, na verdade, nem sequer funciona sem HTML. Caso eu apague todas as tags e deixe apenas o documento, as unicas definições que fiz no estilo padrão que vão permanecer é a cor de fundo, justamente porque a cor de fundo está sobre o body do html.
