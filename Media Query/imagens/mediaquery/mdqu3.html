Media Type

Existem vários tipos de media type, mas os principais são 3: screen, all e print

  • Screen: screen se refere a telas no geral (abrange celular, televisões, computadores...)
  • Print: se refere à impressão, ou seja, como a tela será exibida ao imprimir
  • All: se refere a todos os tipos de tela. É usado quando todos os formatos compartiham de algumas caracteristicas parecidas

Porém, "screen" é amplo demais e não permite alterações para tamanhos diferentes de tela. Para realizar essas personalizações especificas, se usa o media features.

O que é esse "[...] and (orientation: portrait)" e "[...] and (orientation: landscape)"?

Portrait e landscape se referem ao modo de exibição das telas:

  • Portrait são telas em que a altura tem um valor maior que largura
  • landscape são telas em que a largura tem um valor maior que a altura

3 Formas De Usar o Media Query

Existem, então, essas formas de usar o Media Query:

  • 1 Documento CSS para o all, portrait e landscape
  • 1 Documento CSS onde reuna todos, lembrando que, nesse caso, é preciso usar o @media screen e o @media all
  • Sem documentos externos, aplicando todas as alterações em um CSS Local
