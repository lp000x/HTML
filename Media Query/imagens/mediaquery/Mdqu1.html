O Que São Media Queries?

Media queries (no plural) ou Media Query (no singular), são uma forma de deixar um site responsível para diferentes tamanho de tela, de computadores até smartwatches.

A media query faz várias coisas:

Neste exemplo, o menu esquerdo está diferente conforme o tamanho da tela. Para telas grandes, aparecem todos os vídeos e também a barra esquerda com icones, canais e outras informações.

Para telas medianas, aparecem apenas os icones de navegação

Para telas pequenas, aparecem só os 3 traços.

Media Query faz isso: adapta a exibição elementos e recursos e também o tamanho deles conforme o tamanho da tela.

Impressão

Outra coisa que a media query faz é modificar a tela para a impressão (ou seja, se a pessoa quiser ou precisar imprimir em um papel o site)
