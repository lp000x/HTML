O Que É Mobile First?

Mobile first é uma visão que aponta para, primeiro, a criação de um site para dispositivos móveis e depois para web. Porque?

Por que é uma prática muito comum ir "capando" os elementos da pagina apenas para caber tudo em telas menores. De certa forma, isso pode empobrecer a experiência do usuário, uma vez que só conseguirá consumir todos os elementos da página se usar em telas maiores. Um site mais bonito gera mais confiança e credibilidade, pois aparenta ser mais profissional. Além disso, se o usuário tiver uma boa experiência na página ele ficará satisfeito com o site.
