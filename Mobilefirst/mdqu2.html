Typical Device Breakpoint

Typical device breakpoint são tamanhos padronizados de telas, de pequenas (celulares) a grandes (TVs). Esse valor muda constantemente, pois as telas sempre estão se atualizando, então sempre é necessário consultar esses tamanhos.

Os dados que vou colocar abaixo não são do curso em vídeo, uma vez que ele foi gravado em 2020. Os dados abaixo são do Google, desse ano (2026):

  • Extra Pequeno: 320px - 575px
  • Pequeno: 576px - 767px
  • Médio: 768px - 991px
  • Largo: 992px - 1199px
  • Extra Largo:1200px - 1399px
  • Extra Extra Largo: 1400px+

Para que isso serve? Até agora foi mostrado sobre media query e media features.

Media query é a adaptação do site para diferentes telas (de forma bem geral mesmo). Media Features é algo mais especifico: é definir se as estilizações feitas no documento CSS é para todas as páginas ou se é apenas para algumas, sendo esse "algumas" baseado em seu tamanho.

  • Media="all": as configurações feitas com essa media featura serão aplicadas em todas as páginas com link para esse doc CSS
  • Media="print": as configurações serão aplicadas somente em páginas que serão imprimidas
  • Media="screen": as configurações serão aplicadas para as "telas", de modo geral
  • Media="screen and(orientation:landscape)": é uma especificação para o media screen, as estilizações serão aplicadas apenas quando a largura da tela for maior que a altura (pois landscape significa paisagem, e paisagem é a designação para dimensões em que a largura é maior que altura)
  • Meida="screen and(orientatio:portrait)": assim como landscape, é uma especificação para o media screen, porém as alterações feitas serão aplicadas apenas quando a altura for maior que a largura.

Porém, mesmo com as especificações de portrait, ainda fica abrangente, pois uma telva de TV 40 polegadas tem uma largura maior que a altura, da mesma forma que um monitos de 20 polegadas também tem. Para especificar ainda mais, se usam os typical device breakpoint

Para usar é muit simples:

  • Em um documento CSS (ou CSS local), eu uso a regra @media com screen
  • Frente ao screen, adiciono com and e ()
  • Entre parenteses, eu adiciono max-widith e o valor do breakpoint
