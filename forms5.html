Cores

O input também oferece recursos para o usuário escolher cores.

Por padrão, sempre carrega o preto, mas caso eu queira que outra cor seja carregada, basta adicionar o atributo value="" e o código da cor (em hexadecimal, lembrando de colocar a # no começo)

Range

O range é um recurso que permite o usuário medir algo. Por exemplo, as pesquisas de satisfação

Por padrão, o range vai de 0 a 100. Caso eu precise de números menores (ou maiores), basta usar os atriburos min="" e max=""

File

O file permite que o usuário envie arquivos. Existe também a opção image, a diferença é imagem sou eu quem defino, enquanto o file é o usuário que define. Lembrando que, para o envio de arquivos, o método a ser usado é Post, e não Get.

Value

Value é um atributo que pode ser usado em todos (ou quase todos) os recursos de input. Ele serve para definir o valor padrão dos inputs que serão exibidos inicialmente.
