Sandbox

Sandbox é um atributo que gerencia permissões de ações dentro do iframe. Uma forma de garantir a segurança do iframe é bloquear ações com potencial malicioso, como enviar cadastros. Para uso, se usa o atributo sandbox="sandbox", que é o valor de segurança maxima, ou seja, tudo do site externo (scripts, fomulários...) é bloqueado. Com esse atributo definido, eu posso clicar quantas vezes quiser no botão que não vai.

Com esse atributo eu posso permitir algumas coisas:

  • Scripts
  • Formulários
  • Coleta de dados

Referrer Policy

Outra forma de deixar o site mais seguro é usar o atributo referrerpolicy="no-referrer". Esse atributo bloqueia a coleta de dados do usuário dentro do site do iframe.
