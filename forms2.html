Segurança Nos Formulários

Apesar de os dados não estarem indo para lugar nenhum, eles são exibidos na URL do navegador:

Para garantir segurança mais avançada, é preciso estudar temas mais avançados (como protocolos), mas em HTML tem duas formas:

  • Get
  • Post

Para utilizar um desses 2, basta usar o atributo method na tag form

Get

Por padrão, caso eu não defina nada, get é o padrão. Ou seja, é ele quem exibe os dados na URL.

Quando usar get?Get, por exibir os dados no campo de input, acaba se tornando compartilhavel. Caso o input não peça dados sensiveis, o get pode ser utilizado.

Post

Quando utilizo o post, os dados inseridos nos forms não são exibidos na URL. Isso significa que os dados estão seguros? Não completamente. De fato, não estão sendo exibidos na URL, mas sem aplicar criptografia, ainda dá para ver os dados.

Quando usar o Post?O post é uma opção melhor quando se trata de arquivos e dados sensiveis.

Diferença Na Velocidade

Há um tempo atrás, o uso de Get ou Post influenciava na velocidade do site. Hoje ainda influencia, mas os dispositivos evoluiram ao ponto dela ser imperceptível.
