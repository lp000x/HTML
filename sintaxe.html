O Que É Uma Sintaxe

Em portugues, uma sintaxe é:

Sintaxe é o ramo da gramática que estuda como as palavras se organizam e se relacionam em frases e orações para formar sentido, analisando funções como sujeito, predicado, objeto, adjuntos, e a ordem (colocação) das palavras, essencial para a clareza da comunicação

Sintaxe Em Html

Em HTML, as coisas são mais simples. Uma sintaxe é o conjunto de regras que dizem como uma instrução deve ser escrita para que o navegador entenda e interprete a página sem erros.

Estrutura de Uma Sintaxe

As Tags possuem uma sintaxe simples:

  • Toda tag que envolva um conteúdo deve ter uma tag de abertura e outra de fechamento
  • A tag de abertura contém apenas o sinas de maior que (>) e menor que (<)
  • A tag de fechamento, além de conter os sinais maior que (>) e menor que (<) também possui uma barra virada para o direita.
  • No caso de adicionar atributos ou estilização inline, a adição é feita somente na tag de abertua
  • No caso de aninhamento, é preciso manter a tag original por fora.

Aninhamento

Anihamento é quando se adiciona mais de uma tag em torno de um mesmo contéudo. Por exemplo: esta frase está aninhada

Eu usei 3 tag nessa frase, isso é aninhamento.

Case Sensitiva

Algumas tags são case sensitive, ou seja, se eu colocar a tag em letra maiuscula ou minuscula pode dar erro.

Atributos

Atributos são informações extras adicionadas as tags, que alteram ou adicionam instruções novas em seu comportamento. Atributos também tem algumas regas de sintaxe:

  • Como dito antes, sempre são colocados na tag de abertura
  • Todo atributo tem um valor, que é a instrução propriamente dita
  • O atributo segue a regra atributo="valor", ou seja, todo atributo segue um sinal de = e seu valor é colocado entre aspas (""), sem espaços.

Tags E Seus Atributos Padrão

Algumas tags sempre estão acompanhadas de algum atributo, justamente porque precisam de um conjunto de instruções (além da própria tag) para causarem o efeito desejado. A tag em si cria a utilidade, mas precisam de um atributo para que o restante do processo seja possível. Por exemplo:

  • A tag img indica que há uma imagem no conteúdo, porém não diz onde ela está. É por isso que o atributo Href é necessário: para dizer onde essa imagem está.
  • A tag a faz com que um texto adquira a propriedade de hiperlink, mas não diz para onde esse hiperlink aponta. Mais uma vez, o href se torna necessário para fazer esse papel.

Essa lista aumentará com o tempo:

  • img src="" alt=""
  • a href=""
