Configuração Básica de Iframe

A tag do iframe pode ser usado tanto inline como espaçada.

Espaçada

Usando a tag iframe espaçada, eu posso adicionar outras coisas no meio. A primeira configuração é adicionar um link, porque caso a dispositivo não tenha compatibilidade com o iframe, um link será exibido em seu lugar.

OBS: Para o iframe funcionar, ele precisa estar em uma tag p (ou talvez outras tags de texto).

Parametro Novo: Scrolling

Como dá para perceber, tem uma barra de rolagem para exibir páginas com muito conteúdo. Caso eu queria retirar isso, eu posso usar scrolling="no"

  • Scrolling="yes": permite a rolagem
  • Scrolling="no": não permite a rolagem
  • Scrolling="auto": se tiver muito conteúdo, exibe a rolagem. Se não tiver, não exibe.

Frameborder

O Frameborder são as bordas do iframe, e só aceitam os valores de 0 (sem borda), 1 ou 2 (borda de 1px). Caso eu queira adicionar outro tipo de bordar, posso deixar em 0 e adicionar uma borda normal em CSS.
