Adicionando Um Botão

Falta uma coisa nessa formulário: um botão para o usuário enviar seus dados. Para criar um botão, é muito simples: basta usar novamente a tag input, input:submit

  • Type: indicando que o tipo do conteúdo é de texto

Para Onde Os Dados Vão

Os dados vão para uma linguagem de programação. Isso é tema para futuros aprendizados, mas para enviar os dados, eu uso o action="cadastro.java", pode ser java ou qualquer outra linguagem. O action vai ativar quando a pessoa clicar no botão, enviando os dados para onde coloquei entre aspas.

Labels

Os navegadores não sabem que o espaço para colocar os dados estão ligados ao texto atrás deles. Usar um recurso que estabeleça essa ligação ajuda os navegadores a interpretar que existe um formulário no site.

Para criar essa ligação, é utilizada a tag label envelopando apenas a palavra.

O label vem com o atributo for:"", e é nele que eu coloco o identificar que criará a ligação entre texto e caixa de input. Como dito antes, o input vem com 3 atributos, e um deles é o id. É o id que vou colocar no for

Visualmente, parece que nada mudou. Porém agora existe uma ligação entre o texto e seu input, e uma forma de ver que os navegadores "sabem" disso é passando o mouse ou clicando em cima do texto. Nesse ultimo formulário a caixa é selecionada, enquanto que nos outros (nos quais eu não criei label) não acontece nada.
