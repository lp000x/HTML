Tags Sem Fechamento

Algumas tags não precisam de fechamento:

  • br
  • li
  • hr
  • meta
  • link
  • input

Nenhuma delas precisa por um motivo muito simples: elas apenas declaram uma informação. Hr apenas declara uma linha, br apenas declara um espaço entre 2 elementos, link apenas declara uma linkagem e etc. Por isso, essas tags recebem o nome de "void elements" (elementos vazios).

Tags Com Fechamento

A maioria das tags exigem fechamento, também por um motivo simples: essas tags envolvem um conteúdo. Para que exista uma delimitação de até onde a instrução da tag será aplicada, é preciso utilizar uma tag de fechamento.
