IMAGENS DE FUNDO

Como deu para ver nessa ultima imagem, não teve ajuste nenhum, apenas cortou a imagem para caber dentro do espaço da div.

Para adicionar um gradiente, se usa background-image: linear-gradient. Para adicionar uma imagem, se usa background-image: url.

  • Para adicionar gradiente, se usa background-image com "linear-gradient";
  • Para adicionar imagens, se usa background-image com url();

Relembrando...

Cuidado com o tamanho da imagem (em relação ao peso dela), porque imagens pesadas deixam o site lento, ocupam mais espaço no servidor (o que pode custar mais no amontado final de arquivos) e fazem que o site seja penalizado nos resultados de buscas (afinal, ninguém gosta de site lento).
