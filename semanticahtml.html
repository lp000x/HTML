O Que É Semântica

No português, semântica é o estudo do significado das palavras, entendendo como o contexto que são ditas afeta sua interpretação. "Banco" pode ser uma instituição bancária como também pode ser um banco para se sentar.

Semântica em HTML 5

A semântica é uma atualização recente no HTML que faz parte de sua versão 5. No HTML 4 e anteriores, muitas tags eram usadas nas linhas de código, principalmente de estilização. Isso fazia com que as linhas de código ficassem muito poluidas, dificultando a visualização. Com a atualização, certas tags cairam em desuso e, em seu lugar e cumprindo a mesma função, novas tags foram adicionadas, porém com semântica. Em vez de usar divs para criar o cabeçalho, corpo e rodapé de um artigo, por exemplo, agora tem as tags header, main e footer. Isso facilita principalmente na parte de estilização, uma vez que dispensa a necessidade de criar classes ou IDs para aplicar o CSS.

Qual O Seu Propósito

A semântica do HTML vem justamente para isso: deixar o código mais limpo, sem misturar html com CSS. Com isso, vários outras vantagens surgem:

  • Melhora o SEO
  • Aumenta a acessibilidade
  • Aumenta a compreensão do conteúdo.
